## 使用说明

这些图片用于：
1. 教程文档中的插图说明
2. 配置界面的截图展示
3. 网络拓扑的可视化说明
4. 故障排除的流程图解

## 图片格式

- 主要使用PNG格式，支持透明背景
- 截图使用JPG格式，文件更小
- 流程图使用SVG格式，支持缩放
